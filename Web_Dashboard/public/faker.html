OneBonsai — Session Faker

Synthetic VR sessions for grid-view testing

Single Faker

Registers as a fake headset, publishes a canvas-generated video to t-<tenant>-<code>, and prints any incoming commands back into the canvas so you can verify the round-trip.

Idle

Live preview of what the grid view sees as this session's video tile.

Spawn Multiple Sessions

Opens N popups (each a single faker in its own browser context) with pre-canned OneBonsai scenarios. Use this to populate the grid view for demos. Browsers may ask permission to open popups the first time.

Tip: each popup uses its own 4-digit code and is independent. Close any popup to drop that session from the grid.
